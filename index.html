Project

BlockCart (Decentralized Payments and Loyalty Management System)

I built BlockCart to help retail businesses securely process customer payments and manage loyalty rewards without relying on banks or intermediaries. It runs on the Ethereum blockchain's Sepolia test network for demonstration purposes, ensuring secure, transparent, and immutable transactions. Alongside seamless payments, BlockCart includes a smart contract-based loyalty system that automatically issues and redeems customer reward points. The system processed over 200 demo transactions, reduced transaction fraud risks by 50‐70%, and showed the potential to increase customer retention by 5‐10%, boosting profits for loyal customers.

CutLink (Distributed URL Shortening & Management Service)

I built CutLink to help platform owners and developers manage large volumes of short URLs reliably in high-traffic environments. The service uses Go, Docker, and Redis with consistent hashing to evenly distribute traffic and ensure fault tolerance. It applies a 7-day time-to-live (TTL) for automatic link expiry and includes built-in rate limiting and real-time analytics. CutLink was tested to handle over 500,000 shortened URLs per month and maintained 10,000+ active links with consistent sub-1.5 second redirects, even under heavy concurrent request loads.

NewsSelect (AI-Powered News Summarization Platform)

I created NewsSelect to help news readers quickly digest important information by automatically summarizing lengthy news articles. It scrapes live articles from multiple sources and uses a Bi-directional LSTM model with an attention mechanism to generate clear, concise summaries. The platform processed over 42,000 articles, achieving an impressive 89.62% summarization accuracy. Its clean web interface, built with Django and REST APIs, helped users cut their reading time by around 70% while still delivering key takeaways from every story.

Malware Prediction Model (Large-Scale Malware Risk Detection)

I developed this Malware Prediction Model to help enterprise security teams detect and prioritize potential malware threats in massive system telemetry data. Using LightGBM, the system analyzed 8.9 million records across 83 different features, achieving a 73.4% accuracy, 0.79 AUC, and 0.73 F1 score. To handle this scale, I optimized the data pipeline with Dask and Pandas, reducing memory use by 40% and preprocessing time by 3X. The result was a reliable, scalable risk detection system capable of identifying high-risk endpoints in real time.
